# 絲印一鍵生成

這是一個用於嘉立創EDA / EasyEDA 專業版 PCB 編輯器的擴充功能，可依照排針、排母焊盤上的網路名稱快速產生與管理絲印。

## 功能概覽

- 可辨識目前選取的排針或排母元件，也支援從已選取的焊盤回溯到其所屬元件。
- 依照焊盤網路名稱自動提取較短的標籤；若沒有網路名稱，則退回為 `P1`、`P2` 這類腳位編號。
- 支援自訂標籤映射，可逐行填寫 `原標籤=顯示標籤`，例如 `GND=G`、`PA15=A15`。
- 自動分析元件方向、單雙排結構、列分組與外框範圍。
- 提供緊湊型分頁面板，可設定字型、單位、圖層、字號、粗細、相對位置、角度、偏移、外框與反相效果。
- 在布局頁提供效果示意，並盡量縮小面板占用，減少對 PCB 畫布的遮擋。
- 生成時改為先在 PCB 畫布上框選範圍，再依照範圍分布每個絲印，不會把整組絲印堆到同一點。
- 在較密集區域會依照你設定的字號自動微縮部分絲印，以減少相鄰標籤重疊。
- 支援刪除目前排針已生成的整組絲印；舊版本未記錄的絲印會回退為框選範圍刪除。
- 對網路解析與幾何解析加入快取，重複生成同一個排針時等待更短。

## 生成流程

1. 在 PCB 中選取一個排針、排母，或選取其任一焊盤。
2. 開啟頂欄選單 `絲印一鍵生成 > 生成絲印...`。
3. 在「快速」與「布局」頁調整參數；若要縮寫標籤，可在「標籤映射」輸入框中逐行填入規則。
4. 點擊 `生成`。
5. 回到 PCB 畫布，以兩次點選或拖曳方式定義一個矩形分布範圍。
6. 外掛會依照該範圍分布絲印，並在必要時自動縮小密集區域中的標籤。

## 刪除流程

1. 在 PCB 中選取一個已生成過絲印的排針或排母。
2. 開啟同一個參數面板。
3. 點擊 `刪除`。
4. 若該組絲印由目前版本生成，外掛會依照記錄直接刪除。
5. 若該組絲印來自舊版本且沒有記錄，外掛會要求你在 PCB 中再框選一次刪除範圍。

## 標籤映射寫法

在「布局」頁的映射框中逐行填寫：

```text
GND=G
PA15=A15
USART1_TX=TX
```

說明：

- 左側是解析後的原始標籤。
- 右側是最終顯示在絲印上的文字。
- 支援使用 `=`、`:`、`->`、`=>` 作為分隔符。
- 空白行、以 `#` 開頭的行、以 `//` 開頭的行都會被忽略。

## 可設定項目

- 字型：選擇用於絲印文字渲染的字型。
- 數值單位：在 `mil` 與 `mm` 之間切換輸入與顯示單位。
- 圖層：可自動跟隨元件面別，或強制輸出到頂層 / 底層絲印層。
- 字號與粗細：控制文字整體尺寸與筆畫厚度。
- 相對位置與角度：控制文字位於元件哪一側，以及使用自動角度或固定角度。
- 偏移：控制文字與元件本體之間的距離。
- 生成外框：為整組標註增加外框。
- 反相：以反白視覺風格渲染文字。
- 標籤映射：將常用網路名替換成你習慣的縮寫。

## 使用限制

- 僅支援在 PCB 編輯器中執行。
- 一次只處理一個排針或排母；若同時選取多個元件，外掛會提示重新選擇。
- 需要能正確讀取焊盤與網路資料，否則無法產生結果。
- 自訂映射作用於解析後的標籤，而不是直接替換整條原始網路路徑。

## 開發與建置

需求：

- Node.js `>= 20.17.0`
- 嘉立創EDA / EasyEDA Pro 擴充執行環境 `^3.0.0`

本機建置：

```bash
npm install
npm run build
```

建置完成後，`build/dist/` 會產生 `.eext` 擴充套件封包，可直接匯入嘉立創EDA / EasyEDA Pro 安裝測試。

## 專案結構

- `src/index.ts`：擴充入口，負責註冊選單並開啟參數面板。
- `iframe/header-silk.html`：參數面板頁面。
- `iframe/js/header-silk.js`：元件辨識、網路解析、預覽、映射、刪除與絲印生成邏輯。
- `iframe/css/header-silk.css`：面板樣式。
- `build/packaged.ts`：將專案封裝為 `.eext` 擴充檔。
- `locales/`：擴充中介資料與提示文字的多語系資源。
