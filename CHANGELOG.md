# 1.3.1

## 变更

1. 修正排针丝印按框选范围分布时的顺序映射，改为优先按焊盘编号自然顺序生成，避免与排针实际顺序颠倒

# 1.3.0

## 变更

1. 删除之前残留的整组合成、预览确认与旧排布生成逻辑，统一改为参考 `eext-generate-silkscreen` 的逐条丝印生成方式
2. 丝印生成改为严格按框选范围分布，不再把整组内容集中生成到单个点击点
3. 保留当前排针识别与参数面板，但最终落板链路统一为“框选范围后逐项生成文字与外框”

# 1.0.16

## 变更

1. 重新校准丝印字号映射，减少输入字号和实际生成大小的偏差
2. 为文字增加固定上下留边，让排针丝印更容易对齐
3. 提高整组合成时的采样精度，减少最终合成后的尺寸漂移

# 1.0.15

## 变更

1. 重新校准整组合成对象的文字尺寸和粗细换算
2. 合成丝印改为复用实际文字位图，减少最终生成尺寸和面板参数不一致的问题

# 1.0.14

## 变更

1. 校准文字粗细换算，让输入粗细更接近最终生成结果
2. 统一面板示意和实际生成的粗细比例，减少看起来一致但生成后变细的问题

# 1.0.13

## 变更

1. 生成流程改为直接把整组丝印作为一个整体生成到当前鼠标位置
2. 移除放置阶段界面，不再进入“预览/确定/取消”流程
3. 生成后的丝印保持为一个可继续拖动的整体对象

# 1.0.12

## 变更

1. 放置交互改回 `Enter` 确认、`Esc` 取消
2. 确认时按最后一帧预览对应的坐标和尺寸生成最终独立丝印
3. 删除整体预览后再保留最终丝印，避免把预览直接锁成成品

# 1.0.11

## 变更

1. 生成完成后立即退出放置界面，不再停留在“放弃”界面
2. 清除选中状态改为后台执行，避免影响放置收尾速度
3. 保持当前的 `Esc` 完成交互不变

# 1.0.10

## 变更

1. 放置时改为按 `Esc` 直接完成当前预览并锁定
2. 面板中移除可见的“确定”动作，只保留“放弃”用于撤销
3. 完成后自动清除当前选中状态，减少后续干扰

# 1.0.9

## 变更

1. 确认时直接锁定当前预览图元，不再额外生成新的确定丝印
2. 保留最后一帧预览位置作为最终位置，减少确认后的位移风险
3. 生成前继续清理同位置旧内容，避免重复叠加

# 1.0.8

## 变更

1. 放置阶段恢复为单一整体预览，移动鼠标时整体跟随
2. 确认时按最后一帧预览位置生成最终丝印
3. 最终生成的文字和外框默认锁定，避免确认后再移动

# 1.0.7

## 变更

1. 预览阶段恢复为独立预览流程，移动鼠标时先看位置再确认
2. 确认时直接将最后一帧预览对象锁定为最终结果，不再重建
3. 继续减少成功与取消提示，避免频繁打断

# 1.0.6

## 变更

1. 放置阶段改为真实丝印对象整体跟随鼠标，确认时直接保留当前位置
2. 取消放置时删除当前丝印对象，避免再走“预览后重建”的流程
3. 成功、失败和空结果改为轻提示，减少弹窗打断

# 1.0.5

## 变更

1. 外框宽度改为与排针本体宽度一致，不再额外内收
2. 点击确认后直接按最后一帧已显示的预览位置生成，不再按鼠标位置刷新
3. 继续保持预览整体移动、确认后独立生成

# 1.0.4

## 变更

1. 确认时直接按最后一帧预览位置落板，让生成位置与预览位置一致
2. 外框宽度继续收紧，进一步贴近排针本体
3. 保持预览为整体移动，确认后文字和外框仍可分别调整

# 1.0.3

3. 进一步收紧外框宽度，让外框更贴近排针本体

# 1.0.2
