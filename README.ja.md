# シルク一括生成

JLCEDA / EasyEDA Pro の PCB エディタ向け拡張機能です。ピンヘッダやソケットヘッダのパッドに接続されたネット名から、シルク文字をすばやく生成・管理します。

## 主な機能

- 選択中のヘッダ部品を検出し、パッド選択時は親部品まで自動でたどります。
- ネット名から短いラベルを抽出し、ネットが無い場合は `P1`、`P2` のようなピン番号にフォールバックします。
- `GND=G`、`PA15=A15` のような自作ルールを 1 行ずつ書けるラベルマッピング機能を搭載しています。
- 部品の向き、単列 / 複列構造、行グループ、外枠範囲を自動解析します。
- フォント、単位、レイヤー、文字サイズ、線幅、配置方向、角度、オフセット、外枠、反転表示を設定できるコンパクトなタブ式パネルを提供します。
- レイアウトタブにプレビューを表示しつつ、パネル占有面積を小さく保ちます。
- 生成時は PCB キャンバス上で配置範囲を決め、その範囲に沿って各ラベルを分配するため、全ラベルが 1 点に重なりません。
- 密集した領域では、設定した文字サイズを基準に一部ラベルを自動で少し縮小し、重なりを抑えます。
- 選択したヘッダに対して生成済みのシルク一式を削除できます。旧バージョンで記録のないシルクは範囲指定削除にフォールバックします。
- ネット解析と幾何解析にキャッシュを入れているため、同じヘッダでの再生成はより高速です。

## 生成手順

1. PCB エディタで 1 つのピンヘッダ、ソケットヘッダ、またはその任意のパッドを選択します。
2. 上部メニューから `絲印一鍵生成 > 生成絲印...` を開きます。
3. `快速` タブと `布局` タブで設定を調整します。ラベルを短縮したい場合は、マッピング欄に規則を 1 行ずつ記入します。
4. `生成` をクリックします。
5. PCB キャンバスへ戻り、2 点指定またはドラッグで矩形の配置範囲を決めます。
6. 拡張機能がその範囲にラベルを配置し、必要に応じて密集領域のラベルを自動で縮小します。

## 削除手順

1. すでにシルクが生成されているヘッダを PCB 上で選択します。
2. 同じ設定パネルを開きます。
3. `削除` をクリックします。
4. 現行バージョンで生成されたシルクなら、保存済み記録に基づいて直接削除されます。
5. 旧バージョン由来で記録が無い場合は、PCB 上で削除範囲を再指定します。

## ラベルマッピングの書き方

`布局` タブの入力欄に 1 行ずつ記入します。

```text
GND=G
PA15=A15
USART1_TX=TX
```

ルール：

- 左側は解析後の元ラベルです。
- 右側は最終的に表示したいシルク文字です。
- 区切りには `=`、`:`、`->`、`=>` を使えます。
- 空行、`#` で始まる行、`//` で始まる行は無視されます。

## 設定項目

- フォント：シルク文字に使うフォントを選択します。
- 単位：`mil` と `mm` を切り替えます。
- レイヤー：部品面に追従する自動設定、またはトップ / ボトムシルクを固定指定できます。
- 文字サイズと線幅：文字の大きさと太さを調整します。
- 相対配置と角度：部品のどの側に文字を置くか、回転を自動にするか固定にするかを指定します。
- オフセット：文字と部品本体の距離を調整します。
- 外枠生成：ラベル全体を囲む外枠を追加します。
- 反転：反転表示スタイルで文字を描画します。
- ラベルマッピング：よく使うネット名を自分の略称に置き換えます。

## 制限事項

- PCB エディタでのみ動作します。
- 1 回に処理できるヘッダは 1 つだけです。
- 部品からパッド情報とネット情報を取得できる必要があります。
- カスタムマッピングは解析後のラベルに対して適用され、元のネットパス全体を直接置換するものではありません。

## 開発

必要環境：

- Node.js `>= 20.17.0`
- JLCEDA / EasyEDA Pro 拡張ランタイム `^3.0.0`

ローカルビルド：

```bash
npm install
npm run build
```

ビルド後、`build/dist/` に `.eext` パッケージが生成され、JLCEDA / EasyEDA Pro にインポートしてテストできます。

## プロジェクト構成

- `src/index.ts`：拡張機能のエントリーポイント。メニュー登録とパネル起動を担当します。
- `iframe/header-silk.html`：設定パネルの HTML。
- `iframe/js/header-silk.js`：部品解析、ネット解析、プレビュー、マッピング、削除、シルク生成ロジック。
- `iframe/css/header-silk.css`：パネルのスタイル。
- `build/packaged.ts`：`.eext` パッケージを生成するスクリプト。
- `locales/`：拡張メタデータと文言の多言語リソース。
