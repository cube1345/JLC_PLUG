# Генератор шелкографии

Расширение для PCB-редактора JLCEDA / EasyEDA Pro. Быстро генерирует и управляет надписями шелкографии для гребенок и pin / female header на основе сетей, подключенных к их площадкам.

## Возможности

- Определяет выбранный header-компонент и умеет находить родительский компонент по выбранной площадке.
- Извлекает короткие подписи из имен сетей и использует резервные значения вида `P1`, `P2`, если сеть отсутствует.
- Поддерживает пользовательские правила отображения меток по строкам, например `GND=G` и `PA15=A15`.
- Автоматически анализирует ориентацию компонента, одно- или двухрядную структуру, группировку рядов и границы рамки.
- Предоставляет компактную вкладочную панель для настройки шрифта, единиц, слоя, размера текста, толщины линии, стороны размещения, угла, смещения, рамки и инверсии.
- Показывает превью на вкладке компоновки и при этом занимает минимум места на экране.
- Размещает надписи по выбранному прямоугольному диапазону на PCB, а не складывает всю группу в одну точку.
- В плотных местах автоматически немного уменьшает отдельные надписи относительно заданного размера, чтобы уменьшить наложения.
- Позволяет удалить уже сгенерированную группу шелкографии для выбранного header; старые группы без сохраненной записи удаляются через выбор диапазона.
- Использует кэш разбора сетей и геометрии, поэтому повторная генерация для одного и того же header выполняется быстрее.

## Сценарий генерации

1. В PCB-редакторе выберите один pin header, female header или любую его площадку.
2. Откройте верхнее меню `丝印一键生成 > 生成丝印...`.
3. Настройте параметры на вкладках и при необходимости заполните поле сопоставления меток.
4. Нажмите `生成`.
5. Вернитесь на холст PCB и задайте прямоугольный диапазон размещения.
6. Расширение распределит надписи по диапазону и автоматически уменьшит слишком плотные подписи при необходимости.

## Сценарий удаления

1. В PCB-редакторе выберите header, для которого уже была сгенерирована шелкография.
2. Откройте ту же панель.
3. Нажмите `删除`.
4. Если группа была создана текущей версией, она удаляется сразу по сохраненной записи.
5. Если группа пришла из старой версии без записи, расширение попросит выбрать диапазон удаления на PCB.

## Синтаксис сопоставления меток

На вкладке компоновки введите по одному правилу в строке:

```text
GND=G
PA15=A15
USART1_TX=TX
```

Правила:

- Слева находится исходная разобранная метка.
- Справа находится итоговый текст, который должен появиться на шелкографии.
- В качестве разделителя поддерживаются `=`, `:`, `->`, `=>`.
- Пустые строки, а также строки, начинающиеся с `#` или `//`, игнорируются.

## Параметры

- Шрифт: выбор шрифта для текста шелкографии.
- Единицы: переключение между `mil` и `mm`.
- Слой: автоматически по стороне компонента либо принудительно верхний / нижний слой шелкографии.
- Размер и толщина: управляют габаритами текста и толщиной штрихов.
- Относительное расположение и угол: задают сторону относительно компонента и режим поворота.
- Смещение: определяет расстояние между подписью и корпусом компонента.
- Рамка: добавляет контур вокруг всей группы надписей.
- Инверсия: рисует текст в инвертированном стиле.
- Сопоставление меток: заменяет типовые имена сетей на ваши сокращения.

## Ограничения

- Работает только в PCB-редакторе.
- За один запуск обрабатывается только один header.
- Требуется доступ к данным площадок и сетей выбранного компонента.
- Пользовательские сопоставления применяются к уже разобранным меткам, а не к полному исходному пути сети.

## Разработка

Требования:

- Node.js `>= 20.17.0`
- Среда выполнения расширений JLCEDA / EasyEDA Pro `^3.0.0`

Локальная сборка:

```bash
npm install
npm run build
```

После сборки пакет `.eext` появится в каталоге `build/dist/`. Его можно импортировать в JLCEDA / EasyEDA Pro для установки и тестирования.

## Структура проекта

- `src/index.ts`: точка входа расширения, регистрация меню и открытие панели.
- `iframe/header-silk.html`: разметка панели настроек.
- `iframe/js/header-silk.js`: распознавание компонентов, разбор сетей, превью, сопоставление меток, удаление и генерация шелкографии.
- `iframe/css/header-silk.css`: стили панели.
- `build/packaged.ts`: упаковка проекта в файл `.eext`.
- `locales/`: ресурсы локализации для метаданных и подсказок.
